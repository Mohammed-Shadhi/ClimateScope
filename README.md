# ClimateScope: Global Weather Trends & Extreme Events

Developed as part of the Infosys Springboard Data Science Virtual Internship.

This project analyzes global climate datasets to identify long-term temperature trends, seasonal anomalies, and extreme weather patterns using Python and Power BI.

## Technologies Used
- Python
- Pandas
- Power BI
- Data Cleaning
- Exploratory Data Analysis (EDA)
- DAX
