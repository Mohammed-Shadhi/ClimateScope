# ClimateScope: Global Weather Trends & Extreme Events

ClimateScope is a data analytics project developed as part of the Infosys Springboard Data Science Virtual Internship. The project explores global climate datasets to identify long-term temperature trends, seasonal anomalies, and extreme weather patterns through data cleaning, exploratory data analysis (EDA), and interactive Power BI dashboards.

## Key Features
- Preprocessed and cleaned multi-source climate datasets using Python and Pandas.
- Performed exploratory data analysis to uncover climate trends and seasonal variations.
- Identified extreme weather event patterns across multiple regions.
- Built interactive Power BI dashboards with DAX measures for dynamic insights.
- Communicated climate findings through data-driven visualizations.

## Technologies Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Power BI
- DAX
- Jupyter Notebook
- Data Cleaning & Exploratory Data Analysis (EDA)
